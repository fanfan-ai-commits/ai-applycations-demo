# AI扫雷学习游戏 - 规范文档

## 1. 项目概述

- **项目名称**: AI知识扫雷
- **项目类型**: 网页游戏 + 教育应用
- **核心功能**: 经典扫雷玩法结合AI知识问答，踩雷触发AI题目，答对继续游戏，答错扣除生命值
- **目标用户**: 对AI知识感兴趣的学习者

## 2. UI/UX 规范

### 布局结构

- **整体布局**: 单页面应用，居中显示
- **页面区域**:
  - 顶部标题栏：游戏标题 + 生命值进度条 + 分数显示
  - 中部游戏区：雷区网格
  - 底部控制区：新游戏按钮 + 游戏模式说明

### 响应式设计

- 桌面端：雷区格子 40x40px
- 移动端：雷区格子 32x32px
- 最小宽度：320px

### 组件规范

#### 顶部状态栏
- 游戏标题：居中，加粗，发光效果
- 生命值：进度条样式，带百分比显示
- 分数：数字显示，带动画效果

#### 雷区格子
- 未翻开：深色背景，hover时微亮
- 已翻开：浅色背景，显示数字或空白
- 旗帜标记：🚩 图标
- 地雷：💣 图标，踩到时红色高亮

#### AI问题弹窗
- 居中弹出，半透明黑色遮罩
- 卡片样式，圆角，阴影
- 包含：问题、4个选项按钮、倒计时
- 答题结果：绿色/红色提示

#### 游戏结束弹窗
- 显示最终得分
- 重新开始按钮

## 3. 功能规范

### 扫雷核心功能
- **雷区生成**: 随机埋雷，保证第一步不踩雷
- **数字计算**: 计算每个格子周围8个方向的地雷数
- **空白展开**: 点击空白格子自动翻开周围安全区域
- **旗帜标记**: 右键标记/取消标记地雷

### 游戏参数
- 雷区大小: 10x10 (初级难度)
- 地雷数量: 15个
- 初始生命值: 100%
- 答对奖励: +10% 生命值 (最高100%)
- 答错惩罚: -20% 生命值

### AI题目系统
- 使用通义千问API获取AI知识题目
- 题目类型: 选择题 (4个选项)
- 答题时间: 30秒
- 题目缓存: 预加载3道题

### 用户交互
- 左键点击: 翻开格子
- 右键点击: 标记/取消旗帜
- 点击已翻开数字: 快捷标记周围未翻开格子 (当周围旗帜数=数字时)

### 游戏状态
- 进行中: 可以操作
- 暂停: 回答AI问题期间
- 胜利: 翻开所有非地雷格子
- 失败: 生命值归零
