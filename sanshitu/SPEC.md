# 三视图堆叠应用 - 规格文档

## 1. 项目概述

- **项目名称**：三视图堆叠器
- **项目类型**：交互式3D教学工具
- **核心功能**：用户在3D空间堆叠正方体，并查看生成的三视图
- **目标用户**：高中/大学学习立体几何的学生

## 3. 功能规格

### 3.1 核心功能

#### 3.1.1 正方体放置
- 点击3D场景空白区域，在该位置放置一个小正方体
- 正方体大小：1x1x1 单位
- 最多放置 20 个正方体
- 放置时有轻微放大动画

#### 3.1.2 网格吸附
- 3D场景显示参考网格（10x10）
- 正方体自动吸附到整数坐标
- 鼠标悬停时显示吸附预览（半透明绿色）

#### 3.1.3 删除模式
- 切换到删除模式后，点击已放置的正方体可删除
- 删除时有缩小消失动画

#### 3.1.4 撤销/重做
- 记录每次放置/删除操作
- 撤销按钮：回退到上一步
- 重做按钮：恢复被撤销的操作
- 最多保存 50 步历史

#### 3.1.5 清空功能
- 一键清除所有已放置的正方体
- 需要确认提示

### 3.2 三视图功能

#### 3.2.1 查看三视图
- 点击按钮后，右侧显示三视图
- 三个视图并排显示：俯视图、正视图、侧视图
- 视图名称标注在顶部

#### 3.2.2 投影辅助线
- 从3D模型向各视图投射虚线辅助线
- 辅助线颜色：`#64748b`，虚线样式

#### 3.2.3 虚线表示
- 看不见的轮廓线用虚线显示
- 看得见的轮廓线用实线

### 3.3 视角控制

#### 3.3.1 视角预设按钮
- 等轴测视角（默认）
- 俯视图视角
- 正视图视角
- 侧视图视角

#### 3.3.2 鼠标控制
- 左键拖拽：旋转视角
- 滚轮：缩放
- 右键拖拽：平移

### 3.4 快捷键

| 按键 | 功能 |
|------|------|
| 1 | 切换到放置模式 |
| 2 | 切换到删除模式 |
| V | 俯视图视角 |
| F | 正视图视角 |
| S | 侧视图视角 |
| Space | 切换放置/删除模式 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Delete | 清空所有 |

### 3.5 状态显示

- 已放置正方体数量：显示 "已放置: X/20"
- 当前模式：显示 "放置模式" 或 "删除模式"
- 操作提示：根据当前操作显示提示文字

## 4. 技术实现

### 4.1 依赖
- Three.js (通过 CDN)
- TailwindCSS (通过 CDN)

### 4.2 场景结构
- 场景包含：网格平面、坐标轴、已放置正方体组
- 相机：透视相机
- 渲染器：WebGLRenderer，支持抗锯齿

### 4.3 数据模型
```javascript
// 正方体数据
cubes = [
  { id: 1, x: 0, y: 0, z: 0 },
  { id: 2, x: 1, y: 0, z: 0 },

history = [
  { action: 'add', cube: { ... } },
  { action: 'delete', cube: { ... } },
]
historyIndex = 0
```
