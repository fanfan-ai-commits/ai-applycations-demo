游戏规则说明

1. 6人对战：你和5位AI玩家（新手×2、高手×2、大师×1）进行3轮智力较量。

2. 猜测范围：每人每轮猜测一个0-100之间的整数。

3. 目标值计算：本组所有猜测数的平均值 × 2/3为最终目标值。

4. 计分规则：最接近目标值的得6分，其次得5分，以此类推。

5. 游戏顺序：每轮固定顺序：AI新手甲 → AI新手乙 → 你 → AI高手甲 → AI高手乙 → AI大师。

6. 胜负判定：3轮结束后，总分最高的玩家获胜。
