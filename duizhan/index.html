📚 栈 Stack

LIFO 后进先出

栈是一种受限的线性表，只允许在表尾进行插入和删除操作。

栈顶

-

元素个数

0

💻 代码实现

// 入栈
stack.push(element);

// 出栈
stack.pop();

// 查看栈顶
stack[stack.length - 1];

📌 适用场景

  • • 函数调用栈
  • • 括号匹配
  • • 撤销操作
  • • 表达式求值
